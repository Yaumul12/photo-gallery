# Photo Gallery

Photo gallery with lightbox viewer, grid layout, and image upload.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Gallery** (or Skip).

**Step 3:** Click the **Upload Photos** area to select images from your device. Multiple files supported.

**Step 4:** Photos appear in a responsive grid. Hover over a photo to see its name and category.

**Step 5:** Click any photo to open it in a fullscreen lightbox viewer.

**Step 6:** Use left/right arrows (or keyboard arrow keys) to navigate between photos in the lightbox.

**Step 7:** Press Escape or click X to close the lightbox.

**Step 8:** Use the category filter buttons (All, Nature, People, Food, Other) to filter photos.

**Step 9:** Hover over a photo and click X to delete it.

## Features
- Image upload from device
- Responsive grid layout
- Fullscreen lightbox with navigation
- Keyboard navigation (arrows + Escape)
- Category filtering
- Photo deletion
- LocalStorage persistence

## Tech
HTML5 FileReader API + CSS + JavaScript
